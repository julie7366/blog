今天开发博客后台界面时遇到了这个问题，百思不得其解！！多番尝试后发现这是个bug，其解决方案如下：
```javascript
<div class="wrapper">
            <div class="table-content">
                <el-table
                    :data="tableData">
                    //....
                </el-table>
            </div>
        </div>
```
`el-table`的父元素不能是`flex：1`的元素，需要在外面再加一层元素，并且父元素设置成`absolute`，祖父元素设置成`relative`即可
```javascript
.wrapper{
        flex:1;
        width:100%;
        position: relative;
        .table-content{
            position:absolute; width:100%;
        }
    }
```
