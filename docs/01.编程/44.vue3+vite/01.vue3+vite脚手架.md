> vue3+vite脚手架

1. 安装vite
``` js
 npm i -g create-vite-app
```
2. 创建项目
``` js
   create-vite-app <project-name>
   或者
   cva <project-name>
```   
3. 进入项目
```js
  cd <project-name>
```

1. 安装依赖包并启动项目
``` js
npm install
npm run dev

```
